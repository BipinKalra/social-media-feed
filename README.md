# foo-rum - Social Feed Application

A modern social feed application built with React 18, TypeScript, and Tailwind CSS.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- ✅ Dual authentication (modal + pages)
- ✅ Post creation and feed
- ✅ Smooth animations (Framer Motion)
- ✅ TypeScript with strict mode
- ✅ Tailwind CSS styling
- ✅ Fully responsive
- ✅ Accessible (WCAG 2.1 AA)

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router v6
- Vite
